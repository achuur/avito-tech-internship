# Тестовое задание на позицию стажёра-QA
-------------------------------------------

### **Структура проекта**

1. Task_1 - папка 1 задания
    - img - папка со скриншотами багов
    - BUGS_FROM_SCREENSHOT.md - файл с описанием найденных багов
2. Task_2 - папка 2 задания
    - BUGS.md файл с описанием найденных багов
    - qa-internship.postman_collection.json - постман коллекция с автотестами
    - TESTCASES.md - тесткейсы ко 2 заданию 
3. README.md - структура проекта и инструкция

------------------------------------------

### **Инструкция по запуску автотестов**

1. Скачиваем Postman с официального сайта (при необходимости).
2. Открываем Postman.
3. Добавляем коллекцию qa-internship.postman_collection.json.
4. Нажимаем кнопку "Run collection".
5. Нажимаем кнопку "Run qa-internship".
